# Модификаторы {docsify-ignore-all}

?> Количество модификаторов может быть любым, они могут накладываться друг на друга и взаимодействовать между собой.

- постоянные / временные
- таймер с итерациями и без
- тип параметра сущности
- эффект модификатора: добавить, отнять установить

## Жизненный цикл

- постоянные модификаторы удаляют и их отменить нельзя
- есть источник модификаторов и его ключ
- создание и отмена модификаторов происходят одинаково, только при отмене действие противоположенное оригинальному модификатору
- удаление происходит по ключу источника
