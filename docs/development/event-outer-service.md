## Внешние ивенты 

Данный функционал сделан для подписки не `Ecs` частей с `Ecs` частью

### Подписка 
Подписка в сервисе происходит через `Subscribe`, где необходимо указать фильтр следующее
1. фильтр `Ecs`, на которой подписывается функционал для обновление
2. функционал для реагирования на изменение в фильтре
3. удалять ли подписку при смены сцены

### Обработка ивентов
Если есть подписчики, то идёт поиск по фильтру подписчиков, в случае если фильтр не пустой, то вызывается функция обработки ивента из подписчики с `Id` ивента

### Примеры реализации
- `HealthBar`
- `BossGameplayTrigger`
- `FastTravelPoint`
