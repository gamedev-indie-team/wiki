## Способности {docsify-ignore-all}

?> Механизм способности основан на `ECS` и `ООП`

### Способность
 
- для создания новой способности нужно создать `ScriptableObject` с типом `AbilityConfig`
- в `AbilityConfig` можно указать
    - время отката способности
    - список эффектов
- при создании нового эффекта необходимо наследовать от `AbilityEffect`
- для добавление способности можно использовать `AbilityListConverter`

### Эффект способности 

- в `AbilityEffect` можно настроить следующее
    - анимацию (через список)
        - выполнить после анимации, если указана анимация
        - длительность анимации, если указана анимация и нужна длительность отличная от самой анимации
    - использовать таймер
        - таймер

### Анимация

При добавлении новой анимации необходимо сделать следующее
1. в `HumanoidAnimatorController` открыть `AbilityStateMachine`
2. добавить новый `state`
3. добавить условия перехода
4. указать флаг анимации в `AnimatorAbilityState`
