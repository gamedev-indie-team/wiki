## Взаимодействие с интерактивными предметами

На текущий момент момент интерактивные предметы представлены следующими объектами
- двери  
    - дверь, которая препятствует передвижению
    - дверь, которая перемещает по локации
    - дверь, которая перемещает на другую сцену
- точки быстрого перемещения
- сундуки

При взаимодействии с интерактивным объектов создаётся событие с тэгом `EventPressInteractiveObjectTagComponent`, которое в зависимости от объекта обрабатывается по своему

### Настройка

Для тбп можно указать не перемещать мгновенно камеру при перемещении игрока
Для дверей в зависимости от их основной функции можно указать
    - куда она перемещает по локации
    - на какую сцену она перемещает
    - какое у неё начальное состояние(открытая/закрытая)
Для сундука можно указать наполнение
