# Описание

- на текущий момент ИИ включает в себя
    - патрулирование между указанными точка
        - выполнение некоторых действий на этих точка(пример: переход через дверь)
    - следование за противником
    - атаковать противника
    - блокирование ударов противника
    - использование способностей

?> ИИ основан на `ECS` и других механизмов игры

- для включения ИИ у сущности должен быть компонент `AiTagComponent`
- точки для патрулирования указываются в `PatrolPointConverter`
- настройка блокирования указывается в `BlockConfig`
- для ии необходимо настроить `FollowTrigger`
- для точек патруля необходимо настроить `WayPointTrigger`
    - при необходимость можно указывать действие в `WayPoint` для ии
- настройку способности можно прочитать [здесь](../development/ability.md)
